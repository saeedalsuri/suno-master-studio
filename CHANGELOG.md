# Changelog

All notable changes to this project will be documented in this file.

## [6.0.0] - 2026-04-05

### Added
- Initial open source release
- SUNO V5 compliant prompt generator
- 57+ regional music templates
- V5 Logic Core with automatic compliance checking
- Prompt Forge - Generate style anchors and song structures
- Lyric Studio - Write and validate lyrics with structure templates
- V5 Template Engine - Browse, validate, and manage templates
- Regional support:
  - Arabic (Lebanese, Khaleeji, Egyptian, Maghreb)
  - Persian/Iranian (Traditional & Modern)
  - Turkish & Ottoman
  - South Asian (Qawwali, Ghazal, Bollywood, Punjabi)
  - Latin, Rock, Electronic, Country, Orchestral

### Features
- Automatic V5 compliance validation
- Pink Elephant Paradox fixer (negative prompts → positive)
- Soft Blend mode for smooth genre fusion
- Duet mode for dual vocal templates
- BPM calculator with metric modulation
- Artist bypass descriptors

### Tech Stack
- Electron 33
- React 19
- Vite 6
- electron-builder
